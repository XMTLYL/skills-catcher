## 已知限制

### 1. GitHub API 限制

- Code Search 限流较严格（30 次/小时）
- 搜索结果最多 1000 条
- 只搜索默认分支

### 2. 解析限制

- 依赖 SKILL.md 格式规范
- Fallback 解析可能不准确
- 不支持非标准格式

### 3. 覆盖范围

- 只索引公开仓库
- 可能遗漏未被 GitHub 索引的仓库
- 依赖搜索关键词的准确性
